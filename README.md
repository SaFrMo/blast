## Blast (2.1.0)

Vanilla version of Blast.js.

**Docs**  
[velocityjs.org/blast/](http://velocityjs.org/blast/)

**Quickstart**

```sh
npm install blast-vanilla
```

```js
import Blast from 'blast-vanilla'

new Blast('.selector', { /* options */ })
```
